## 📦 Run Locally

Clone the repository with:

```shell
git clone https://github.com/abdibrokhim/neuronostalgia-hack24
```

Install the dependencies with:

```shell
npm install
```
and 

Run the development server with:

```shell
npm run dev
```
Open http://localhost:3000 in your browser.
